Assessment Methodology
Scoring Approach:
  • Weighted Questions: Each question has a specific weight based on its criticality
  • Stage-based Analysis: Three critical phases of ransomware incidents
  • Percentage Scoring: 0-100% scale for easy interpretation
  • Risk Categorization: Clear readiness levels from Poor to Excellent
Readiness Levels:
Excellent 80-100%
Good 60-79%
Fair 40-59%
Poor 0-39%
Three-Stage Assessment Framework
Pre-Infection Stage

Prevention measures, security controls, user training, and proactive defense mechanisms.

  • Email security
  • Endpoint protection
  • Network segmentation
  • User awareness
  • Access controls
Active Infection Stage

Detection capabilities, incident response procedures, and immediate containment measures.

  • Threat detection
  • Incident response
  • Network isolation
  • Communication plans
  • Evidence preservation
Post-Infection Stage

Recovery procedures, business continuity, and lessons learned implementation.

  • Backup recovery
  • System restoration
  • Business continuity
  • Forensic analysis
  • Improvement planning
