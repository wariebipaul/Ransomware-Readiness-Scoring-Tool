Help & Documentation

Comprehensive guide to using the Ransomware Resilience Assessment Tool effectively.

Quick Start Guide
Getting Started:
  1. Click "Start Assessment" from the dashboard
  2. Enter your organization details
  3. Answer questions across three stages
  4. Review your results and recommendations
  5. Export your report for sharing
Tips for Best Results:
  • Answer all questions honestly
  • Use comments to provide context
  • Consult with technical teams when needed
  • Review recommendations carefully
  • Plan implementation timelines

Detailed User Guide
Assessment Process:
1. Organization Setup

Provide organization name, assessor information, and role. This helps contextualize the assessment and provides proper attribution in reports.

2. Stage Navigation

Navigate between the three assessment stages: Pre-infection, Active Infection, and Post-infection. Each stage focuses on different aspects of ransomware readiness.

3. Question Responses

Select the most appropriate response for each question. Scores range from 0-4 points based on the level of protection or capability described.

Understanding Results:
Overall Score

Weighted average across all stages, presented as a percentage with corresponding readiness level (Poor, Fair, Good, Excellent).

Stage Performance

Individual scores for each of the three stages, helping identify specific areas of strength and weakness.

MITRE Coverage

Analysis of how well your organization addresses specific MITRE ATT&CK techniques commonly used in ransomware attacks.

Troubleshooting:
  • Progress not saving: Ensure JavaScript is enabled and cookies are allowed
  • Charts not displaying: Try refreshing the page or using a different browser
  • Export not working: Check if downloads are blocked by your browser
  • Session lost: Browser session storage may have been cleared

  • Close unnecessary browser tabs
  • Ensure stable internet connection
  • Use latest browser version
  • Clear browser cache if experiencing issues
